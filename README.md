# 📚 Trabalho em Grupo – Lógica de Programação – Portugol | Serratec

Repositório criado para armazenar a resolução das atividades propostas no curso **Serratec**, utilizando a linguagem **Portugol**.

O objetivo do trabalho é praticar conceitos fundamentais de programação como:

- Estruturas de repetição
- Estruturas condicionais
- Vetores e matrizes
- Funções
- Recursão
- Manipulação de cadeias (strings)

Cada integrante do grupo ficou responsável por resolver algumas das questões propostas.
